# AI百宝箱 - 执行计划

## 定位：导航站 + 一个核心工具

### 导航站（已完成）
- 44个外部工具链接，8分类
- 深空玻璃体设计
- 引流层，不需要盈利

### 核心工具：AI 文本助手
- 翻译 + 润色 + 总结 + 改写
- 一个页面，四个模式，一个API
- 3次/天免费 → 用户自带Key
- 纯前端JS直接调DeepSeek API

## 技术方案
- 深空C风格保持不变
- 工具内嵌在导航站中（点击卡片打开工具面板）
- API: 火山引擎ARK的DeepSeek模型
- 免费额度：3次/天（用网站Key）
- 无限额度：用户填自己的API Key
- 关键：UX要丝滑，不输同类产品

## 迭代路径
1. 先做翻译功能（最常用）
2. 再加润色/改写/总结
3. 导航站持续补充新链接
4. 流量起来后考虑私域引流
